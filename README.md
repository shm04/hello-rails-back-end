# ğŸ“– Hello Rails Backend <a name="about-project"></a>

Hello Rails Back-end is a tutorial designed to guide you through the process of creating a Rails API paired with a React Front-end. This project is an integral component of the Microverse curriculum. The objective is to develop a Rails API and React Front-end capable of showcasing a randomly selected greeting retrieved from the database.

## ğŸ’» Getting Started <a name="getting-started"></a>

## To get a local copy up and running, follow these steps.

### Prerequisites

To run this project you need:

- A PC
- Install Ruby
- Install Rails
- Install Postgresql
- Understanding of Ruby

### Frontend

Here is the [Front-End](https://github.com/shm04/hello-react-front-end) of this app.

### Setup

- Clone this repository to your desired folder:

```sh
 git clone https://github.com/shm04/hello-rails-back-end.git
```

- Navigate into the cloned folder

```sh
 cd hello-rails-back-end
```

### Install

- Install this project with:

```sh
 bundle install
 yarn install
```

- Set up the database
  Change the username and password of your postgres account in `config/database.yml`

```sh
 rails db:create db:migrate db:seed
```

- Start the server

```sh
 rails server
```
